### T001: 安全性重构 - JWT Secret外部化
**优先级**: 🔴 高 | **预估工时**: 4小时

#### 任务描述
将JWT Secret从硬编码改为环境变量配置，提高安全性。

#### 具体工作
1. 修改 `JwtUtil.java`，使用 `@Value` 注入secret
2. 更新各服务的 `application.yml` 配置文件
3. 添加环境变量说明文档
4. 验证配置生效

#### 涉及文件
- `common/src/main/java/com/example/common/util/JwtUtil.java`
- `service-*/src/main/resources/application.yml`
- `.env.example` (新增)

#### 验收标准
- [ ] JWT Secret可通过环境变量配置
- [ ] 默认值安全合理
- [ ] 各服务启动正常
- [ ] Token生成和验证功能正常

#### 依赖关系
- 无

3. 更新任务状态为"进行中"

### 2. 任务执行
1. 按照任务说明实施
2. 及时提交代码
3. 记录遇到的问题

### 3. 任务验证

2. 代码审查
